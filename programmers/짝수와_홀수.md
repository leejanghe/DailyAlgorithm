<br />

# 제한 조건

num은 int 범위의 정수입니다.
0은 짝수입니다.

<br />

# 입출력 예

| num | return |
| :-: | :----: |
|  3  |  Odd   |
|  4  |  Even  |

<br />

# 문제 풀이

## Javascript

```js
function solution(num) {
  return num % 2 ? "Odd" : "Even";
}
```
